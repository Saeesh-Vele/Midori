# Security Policy

## Supported Versions

We currently support the latest version of EcoFy.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability, please do NOT create a public GitHub issue.

Instead, please send an email to `security@ecofy.app` or report it via private messaging on our dev platform.

We will acknowledge your report within 48 hours and provide an estimated timeframe for a fix.

Thank you for helping keep EcoFy safe!
